# ERP Frontend Web Application

기업 내부 업무 프로세스를 웹 기반으로 관리하기 위한  
**ERP(Enterprise Resource Planning) 프론트엔드 애플리케이션**입니다.

백엔드 API 구조를 고려하여  
**인증 흐름, 권한 제어, 화면 단위 역할 분리**를 중심으로 설계했습니다.

---

## Tech Stack

**Frontend**
- React 18
- TypeScript
- Vite

**UI / Styling**
- Styled-components
- SCSS

**Auth / Integration**
- JWT 기반 인증 연동
- 권한 기반 라우팅 처리

**Development**
- ESLint
- Prettier

---

## Key Features

- 로그인 및 회원 관리 화면 구현
- ERP 대시보드 UI 구성
- 재고 관리 화면 및 CRUD UI 설계
- JWT 인증 상태 기반 접근 제어
- 권한(Role)별 화면 접근 제한

---

## Portfolio Highlights

- 백엔드 인증/인가 구조를 고려한 프론트엔드 설계
- 화면 단위 컴포넌트 분리 및 재사용성 확보
- TypeScript 기반 타입 안정성 확보
- 실제 ERP 시스템을 참고한 화면 레이아웃 구성
